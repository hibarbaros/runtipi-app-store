# Shopware 6

Shopware 6 is a modern, open source e-commerce platform based on Symfony Framework and Vue.js. It provides a flexible and scalable solution for online stores of all sizes.

## Features

- Modern Technology Stack: Built on Symfony 7 and Vue.js 3
- Headless Commerce: API-first architecture for flexible frontend implementations
- Extensible: Over 3,100 community extensions available
- Flexible: Plugin and App system for customization
- Open Source: MIT licensed, completely free

## Installation

After installing Shopware 6 through Runtipi, you can access:

- Storefront: Access your shop at the configured domain
- Admin Panel: Access the admin panel at /admin
  - Default credentials: admin / shopware

## Requirements

- MySQL 8.0 database
- PHP 8.2+
- Minimum 2GB RAM recommended
