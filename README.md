# Runtipi App Store - Shopware 6

This is a custom Runtipi app store containing Shopware 6 application.

## Shopware 6

Shopware 6 is a modern, open source e-commerce platform based on Symfony Framework and Vue.js.

## How to Add This App Store to Runtipi

1. Go to Runtipi Settings → App Stores
2. Click "Add App Store"
3. Enter the following:
   - **Name**: shopware6-store (or any name you prefer)
   - **URL**: `https://github.com/YOUR_USERNAME/runtipi-app-store` (replace with your GitHub username)
4. Click "Add"

After adding, Shopware 6 will appear in your App Store list and can be installed like any other app.
